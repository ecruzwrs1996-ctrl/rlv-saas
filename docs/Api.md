# API Documentation

## Overview

The RLV-SaaS backend exposes a RESTful API developed with CodeIgniter 4.

Base URL:

```text
http://localhost:8080/api
```

Authentication is handled using JWT tokens.

---

## Authentication

### Login

```http
POST /api/auth/login
```

Request:

```json
{
  "username": "SysAdm",
  "password": "********"
}
```

Response:

```json
{
  "success": true,
  "token": "jwt-token",
  "user": {}
}
```

---

## Dashboard

### Statistics

```http
GET /api/dashboard/stats
```

Returns dashboard metrics and summary information.

### Recent Activity

```http
GET /api/dashboard/activity
```

Returns recent system activity.

### Activity Timeline

```http
GET /api/dashboard/activity-timeline
```

Returns activity grouped by date and time.

## Users

### Get Users

```http
GET /api/users
```

### Get User By ID

```http
GET /api/users/{id}
```

### Create User

```http
POST /api/users
```

### Update User

```http
PUT /api/users/{id}
```

