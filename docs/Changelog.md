# Changelog

All notable changes to this project are documented in this file.

## Version 1.0.0 - Initial Stable Release

Release Date: June 2026

### Added

* Full Dockerized architecture.
* Vue.js frontend application.
* CodeIgniter 4 REST API backend.
* MySQL 8.4 LTS database container.
* JWT authentication system.
* Role-Based Access Control (RBAC).
* User management module.
* Role management module.
* Audit logging module.
* Active users monitoring.
* Dashboard analytics.
* Database migrations.
* Database seeders.

### Infrastructure

* Docker Compose environment.
* Backend Dockerfile.
* Frontend Dockerfile.
* Development and production parity.

### Security

* Password hashing with BCRYPT.
* JWT token validation.
* Audit trail generation.
* Session tracking.

### Database

* Migration-based schema management.
* Seeder-based initialization.
* Linux case-sensitive table compatibility.
* Standardized PascalCase table naming convention.
